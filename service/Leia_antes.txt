A "Api.js" se refere a API de validação e edição destes dados

A "API_Network.js" se refere a API relacionados a rede social, incluindo post, comentários, curtidas e etc.

Os módulos necessários são o Express, Express-Session, BodyParser e Mysql.

Execute este comando antes

npm install express express-session body-parser mysql cors jsonwebtoken

alterar configurações de conexão com o banco de dados

alterar nome da tabela de validação na VARIAVEL

estar conectada na mesma rede, usando o ip nas rotas para funcionar
